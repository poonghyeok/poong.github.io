### [백준 1835 : 카드 ](https://www.acmicpc.net/problem/1835)

#### 제출 (1트 성공)
```python
n = int(input())
indices, card_nums = list(range(n)), list(range(n))

for i in range(n) :
    for _ in range(i+1) :
        indices.append(indices.pop(0)) 
    
    idx = indices.pop(0)
    card_nums[idx] = i + 1

print(" ".join([str(n) for n in card_nums]).strip())
```

#### 참고답안
```python
from collections import deque

n = int(input())
dq = deque()
dq.append(n)  # 마지막 카드를 먼저 넣음

for i in range(n-1, 0, -1):
    dq.appendleft(i)  # i번째 카드를 앞에 넣음
    for _ in range(i):
        dq.appendleft(dq.pop())  # 뒤에 있는 카드를 앞에 옮김

print(' '.join(map(str, dq)))
```
> 덱을 이용한 역재생 구현인데 이해하기 어려웠다.
1. **collections, deque**
`collections` 모듈과 `deque` 클래스 사용
`append` `appendleft` `pop` 등의 메서드 사용
2. `type converting`은 `list comprehension` 보다는 `map(type, obj)` 사용 `map`도 `string.join()`의 `arg`가 될 수 있다.
3. **덱(deque)에 대한 상세설명**
`double-ended queue`로 양 쪽 끝에서 값 `append`, `pop` 둘 다 가능
```python
from collections import deque

dq = deque([1,2,3,4])
dq.pop() # 1,2,3
dq.popleft() # 2,3 
dq.append(1) # 2,3,1
dq.appendleft(4) #4,2,3,1
print(" ".join(map(str, dq))) # 4 2 3 1
```
